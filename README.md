# Importador de Transacciones CSV
Programa en Java que lee un archivo CSV con movimientos financieros y genera un resumen.
## Acciones
-Procesa registros de ingresos (IN) y egresos (OUT)

-Valida datos

-Calcula totales

## Ejecuta
-Registros válidos e inválidos

-Total de ingresos

-Total de egresos
