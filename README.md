# 📊 CSV-IMPORTER-B
### Estudiante: B1_Nayib de la Cruz Márquez

Sistema de importación de transacciones financieras desarrollado en Java.

## 🚀 Funcionalidades
- Procesamiento de archivos CSV con montos monetarios (`BigDecimal`).
- Validación de seguridad contra ataques de Path Traversal.
- Reporte automático de totales (Entradas/Salidas).

## 🛠️ Ejecución
Para correr las pruebas manuales:
1. Configurar la variable `IMPORT_BASE_DIR` con la ruta de tus archivos.
2. Ejecutar la clase `TestManual.java`.
