📍 Importer con Manejo de Excepciones Personalizado. B21_Nayib de la Cruz Marquez 📍 
-Manejo de Errores: Se implementó la clase personalizada ImportException para capturar errores específicos durante la lectura del CSV.
-Seguridad: Validación de rutas para evitar ataques de Path Traversal.
-Lógica de Negocio: Procesamiento de transacciones con cálculo automático de totales (Ingresos: 750.00 / Egresos: 100.00).
-Arquitectura: Separación de responsabilidades en clases Importer, Transaction, Money y ImportResult.
-Configuración: El proyecto requiere la variable de entorno IMPORT_BASE_DIR apuntando a la carpeta data.
